# Mediapipe_multiHandtracker
MultiHandtracker using Mediapipe models in python

# Getting Started
Requirements
------------
* These are required package to use this project
```
numpy
opencv-python
tensorflow
matplotlib
open3d
```

* run
```
$ python run.py
```
